# Porsaanfilettä kermakastikkeessa

### Ainekset
- öljyä
- nokare voita
- **1** porsaan sisäfile
- soijaa
- **1** iso sipuli
- **3 rkl** vehnäjauhoja
- **1/2 l** lihalientä
- **1 dl** kermaa
- valkopippuria
- mustapippuria
- currya
- **2 tl** Dijon-sinappia

### Ohjeet
Voitele file soijalla, anna marinoitua pari tuntia. Leikkaa file viipaleiksi (1,5 cm paksuiksi) ja paista öljy-voiseoksessa pienissä erissä. Laita kaikki viipaleet takaisin pannuun, lisää viipaloitu sipuli ja kuullota. Lisää vehnäjauhot ja sekoita, kunnes ne saavat vähän väriä. Lisää lihaliemi, kerma ja mausteet. Anna hautua 20 minuuttia - kastike saa kiehua vähän kasaan. Säädä mausteilla ja nesteellä tai Maizena-suuruksella haluamaksesi. Itse pidän paksuista kastikkeista, joten lisäilen usein Maizenaa.
